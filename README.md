# Hospital-Operations-Patient-Risk-Analysis
Developed an end-to-end Healthcare Analytics project using Python, SQL, Pandas, NumPy, and Matplotlib to analyze patient readmission risk, treatment costs, hospital operations, and follow-up compliance. Performed data cleaning, EDA, feature engineering, and generated actionable business insights.
